# My Awesome Server

My Awesome Serverはドキュメントの例を示すためのサンプルプロジェクトです。


## Features

- ホットリロード
- 静的ファイルの配信


## Requirements

- Node.js 8.x+


## Installation

```sh
npm install my-awesome-server
```


## Usage

```sh
my-awesome-server
```

[Command](Command.md)


## Configuration

`config.json` に設定を定義できます。

```json
{
    "name": "My Awesome Server",
    "root": "src"
}
```

[Configuration](Configuration.md)
