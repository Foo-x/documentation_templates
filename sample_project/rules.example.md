# 運営ルール

## 課題管理

- 登録
    - 登録できるメンバー
        - foo
        - bar
    - 設定する属性
        - 必須
            - タイトル
            - 担当者
            - 完了条件
        - 推奨
            - 詳細
            - 目的
            - 期限日
            - 優先度
            - カテゴリ
            - タグ/ラベル
- 更新
    - 更新できるメンバー
        - foo
        - bar
    - 更新するタイミング
        - 必要になったとき
    - ステータスの変更
        - 着手可能
            - `Ready`
        - 着手中
            - `In progress`
        - レビュー中
            - `In review`
        - 保留中
            - `Pending`
        - 完了
            - `Done`
- 完了
    - 完了できるメンバー
        - foo
        - bar
    - 完了するタイミング
        - 定例内
    - 完了理由を設定する
        - 対応済み
        - 対応しない
        - 無効
        - 重複
