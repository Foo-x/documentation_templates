# My Awesome Project

My Awesome Projectは開発時に使用するドキュメントのテンプレートを作成するプロジェクトです。

プロジェクトを立ち上げてから閉じるまでに様々なドキュメントを作る必要があります。  
このプロジェクトの目的は、それらのコストを削減するためのテンプレートを用意することです。  
なお、成果物としてのドキュメントは対象としていません。
